<h2>📝 Descripción del proyecto</h2>
Este proyecto consiste en la creación de un **Catálogo de Libros: LiterAlura**.  
El objetivo es consumir una API de libros, analizar datos en formato JSON, guardarlos en una base de datos y mostrar resultados filtrados a los usuarios.  

La aplicación ofrece interacción textual a través de la consola, con al menos 5 opciones de interacción para que el usuario pueda buscar libros y autores de interés.  
Durante el desarrollo se aprenderá a configurar un ambiente en Java, realizar solicitudes HTTP, manipular respuestas JSON, trabajar con bases de datos y presentar la información de forma clara y accesible.

<h2>⚙️ Funcionalidades</h2>

- **Configuración del ambiente Java:** Preparar el entorno de desarrollo para ejecutar el proyecto.  
- **Creación del proyecto:** Generar la estructura inicial en Java.  
- **Consumo de la API de libros:** Realizar solicitudes HTTP para obtener información actualizada.  
- **Análisis de la respuesta JSON:** Interpretar y manipular los datos recibidos.  
- **Inserción y consulta en la base de datos:** Guardar los libros y autores y permitir búsquedas posteriores.  
- **Exhibición de resultados:** Mostrar los datos filtrados de manera clara en la consola.  

<h2>🔗 ¿Cómo pueden usarlo los usuarios?</h2>

Actualmente se está trabajando en la implementación final para que el proyecto esté disponible como **aplicación de consola en Java**.  
El usuario podrá interactuar con el catálogo mediante opciones de texto y explorar libros y autores directamente desde la API.
